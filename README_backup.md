# RAG Chatbot

A local Retrieval Augmented Generation (RAG) chatbot using:

- Sentence Transformers
- ChromaDB
- Hugging Face Transformers
- FLAN-T5


## Architecture

User Question

↓

Vector Search (ChromaDB)

↓

Relevant Context

↓

Prompt

↓

FLAN-T5

↓

Answer


## Features

- Semantic Search
- Vector Database
- Local LLM
- RAG Pipeline
- Context-based answers
